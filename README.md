# TetrisMultiplayer

To play the game on your own machine :

1. Download the code from this repository.
2. Install the required dependencies by running `pip install -r requirements.txt`.
3. Get the IP of the machine you're lauching the server on and replace it in the `config.json` in "server_ip".
4. Launch the `server-main.py` script in the server folder.
5. Then, launch the `main.py` script in the game folder.

You can only play between machines that are connected on the same network 

## Features : 

- Every player have the same blocks in the same order
- When playing with only 2 players, your opponent's screen is displayed on your screen
- You can create your own profile and customize your key binds
- There are different game modes that you can play
